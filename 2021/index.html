For the third time, basel.codes cordially invites you to this year's Processing Community Day. A free, one-day event to celebrate art, code and diversity. We warmly welcome all people who are interested in using code for creative and experimental projects.


All talks & workshops are for free!


Important: Due to the current pandemic, a valid Covid-19 certificate is required to participate physically in these events. There is no opportunity for on-site testing. We plan to stream our talks virtually, stay tuned for details.
